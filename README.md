# bookmark-manager# Bookmark Manager

ポータブルなブックマーク編集ソフト(Tauri 2 / Rust + バニラ JS)。
インストール不要・単一 exe(約 4.2 MB)で動作し、データはブラウザ互換の HTML 形式で保存します。

![構成](https://img.shields.io/badge/Tauri-2-blue) ![サイズ](https://img.shields.io/badge/binary-4.2MB-green)

## 特徴

- **インストール不要のポータブル設計**
  exe 単体で動作。データ (`bookmarks.html`)・設定 (`settings.json`)・アイコンキャッシュ (`icons/`) はすべて exe と同じフォルダに保存されます。レジストリや AppData は使いません。
- **ブラウザ互換のデータ形式**
  ブックマークデータは Netscape ブックマーク形式 (HTML) で保存。Chrome / Edge / Firefox がエクスポートした HTML をそのまま「開く」ことができ、本ソフトで保存した HTML をブラウザにそのままインポートできます。
- **エクスプローラー様式の UI**
  左にフォルダツリー、右にリストビュー。ダブルクリックで開く、ホバーで編集/削除ボタン表示、ダーク/ライトモード自動切替。
- **ドラッグ & ドロップ**
  ブックマークやフォルダをドラッグして、フォルダへの移動も並び順の入れ替えも可能。並び順はファイルにそのまま保存されます。
- **ファビコン表示(オプション)**
  サイトの favicon を 32x32 PNG に統一してローカルにキャッシュ。画面に見えている行だけを遅延読み込みし、OFF にするとキャッシュを即座に全削除します(既定: OFF)。

## 使い方

`bookmark-manager.exe` を好きなフォルダに置いて起動するだけです(初回起動時、exe の隣に `bookmarks.html` などが作られます)。

### ツールバー

| ボタン | 機能 |
|---|---|
| 開く | ブックマーク HTML を選んで読み込み。以後の自動保存先になる(タイトルバーに表示) |
| 保存 | 現在のデータを名前を付けて保存。以後の自動保存先になる |
| ＋ ブックマーク | ブックマークを追加(名前 / URL / フォルダ指定) |
| ＋ フォルダ | フォルダを追加(名前 / 親フォルダ指定) |
| 検索ボックス | 全フォルダ横断で名前・URL を部分一致検索 |
| アイコンを表示 | ファビコンの表示 ON/OFF。OFF にするとキャッシュを全削除(確認あり) |

※ 編集内容は常に現在のファイルへ自動保存されるため、明示的な保存操作は不要です。
「保存」は別ファイルへ書き出したいときに使います。

### マウス操作

| 操作 | 動作 |
|---|---|
| ダブルクリック(ブックマーク) | OS 既定のブラウザで URL を開く |
| ダブルクリック(フォルダ) | フォルダを開く |
| 行ホバー → ↗ / ✏️ / 🗑️ | ブラウザで開く / 編集 / 削除 |
| ドラッグ → フォルダ行・ツリー項目の中央 | そのフォルダの中へ移動(ハイライト表示) |
| ドラッグ → 行の上端 / 下端 | その前 / 後ろに挿入 = 並べ替え(挿入ライン表示) |
| ドラッグ → ツリーの「🏠 ブックマーク」 | ルート直下へ移動 |
| ドラッグ → リストの空白部分 | 表示中フォルダの末尾へ移動 |

自分自身や子孫フォルダへの移動は自動的に禁止されます。

### ブラウザとの連携

- **ブラウザ → 本ソフト**: ブラウザのブックマークマネージャーから HTML をエクスポートし、本ソフトの「開く」で読み込む
- **本ソフト → ブラウザ**: 「保存」で HTML を書き出し、ブラウザのブックマークマネージャーからインポートする

## データファイル

| ファイル | 形式 | 内容 |
|---|---|---|
| `bookmarks.html` | Netscape ブックマーク形式 | ブックマーク・フォルダ階層・並び順 |
| `settings.json` | JSON | アイコン表示設定など |
| `icons/*.png` | 32x32 PNG | ファビコンキャッシュ(ホスト名由来のファイル名) |

旧形式 (`bookmarks.json`) のデータがある場合、初回起動時に自動で `bookmarks.html` へ移行されます。

## ビルド

Rust (stable) が必要です。フロントエンドは静的ファイルのため Node.js は不要です。

```powershell
cd src-tauri
cargo build --release
# 生成物: src-tauri\target\release\bookmark-manager.exe
```

この PC では MSVC なしの GNU ツールチェーン構成でビルドします:

```powershell
$env:Path = "$env:USERPROFILE\.cargo\bin;C:\_ClaudeCode\tools\mingw64\bin;$env:Path"
cd src-tauri
cargo build --release
```

- Windows では WebView2 ランタイムを使用(Windows 10/11 には標準搭載)
- macOS / Linux でも同じソースからビルド可能(Tauri 対応プラットフォーム)

詳細な設計・実装情報は [仕様書.md](仕様書.md) を参照してください。
